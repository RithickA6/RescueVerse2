# RescueVerse AI Development Rules

Project:
- MERN Stack
- React + Vite
- Phaser 3

Rules:

- Never rewrite the project from scratch.
- Always modify existing files.
- Reuse existing architecture.
- Do not duplicate code.
- Keep scenes modular.
- Keep managers reusable.
- Use ES Modules.
- Never break existing gameplay.
- Verify imports after changes.
- Verify project builds successfully.
- Use sprites when assets exist.
- If assets are missing, create documented placeholders.
- Explain every file modified.
- Make changes one milestone at a time.
