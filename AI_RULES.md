# RescueVerse AI Development Rules

Project:

- MERN Stack
- React + Vite
- Phaser 3

Development Rules:

- Never rewrite the project from scratch.
- Modify existing files only.
- Preserve the existing architecture.
- Reuse components and managers whenever possible.
- Do not duplicate logic.
- Keep Phaser scenes modular.
- Do not break existing gameplay unless requested.
- Keep all code clean and production-ready.
- Verify imports after every change.
- Verify the project builds successfully after modifications.
- Explain every modified file after completing a task.

Workflow:

- Complete one milestone at a time.
- Wait for approval before starting the next milestone.
- Prefer reusable systems over scene-specific code.
- Use placeholders only when required because assets are missing.
